# Data Analytics Portfolio

Welcome to my data analytics portfolio. This repository showcases professional data analysis projects and automated workflows specifically tailored for **Construction, HSE, QA/QC, Document Control, and Aconex** management.

The goal of these projects is to transform raw operational data into actionable insights, optimize tracking systems, and improve decision-making efficiency.

---

## 🛠️ Technologies Used
* **Data Visualization:** Power BI, Microsoft Excel (Advanced)
* **Process Automation:** Power Automate
* **Data Management:** SQL, Excel Power Query
* **Domain Tools:** Aconex, Document Control Systems

---

## 📁 Portfolio Projects

### 1. Document Control & Aconex Tracking Dashboard
* **Description:** A dynamic dashboard designed to track submittals, RFIs, and transmittals status across multiple projects.
* **Key Features:** Monitors delayed responses, outstanding document workflows, and consultant review cycle times.
* **Impact:** Reduced document bottleneck identification time by 30% and enhanced team accountability.

### 2. HSE (Health, Safety, and Environment) Performance Report
* **Description:** An interactive safety dashboard analyzing incident rates, near-misses, and safety induction frequencies on site.
* **Key Features:** Heatmaps for high-risk zones, monthly trend analysis, and safety milestone tracking.
* **Impact:** Provided management with immediate visibility into site safety compliance, helping prevent recurring hazards.

### 3. QA/QC Inspection & NCR Analytics
* **Description:** Analysis of Non-Conformance Reports (NCRs) to identify structural or systemic quality issues in materials and execution.
* **Key Features:** Root-cause analysis breakdowns, closed vs. open NCR ratios, and subcontractor performance evaluation.
